# 📚 Study Pomodoro Timer

A clean, minimal Pomodoro timer built with **HTML, CSS, and JavaScript** to help students stay focused during study sessions.

## 🔗 Live Demo
*(Add your GitHub Pages link here after deploying)*

## ✨ Features
- ⏱️ Three modes: Focus (25 min), Short Break (5 min), Long Break (15 min)
- ▶️ Start, Pause, and Reset controls
- 📊 Tracks completed focus sessions
- 🎨 Clean, responsive UI with smooth transitions
- 🔔 Browser alert notification when a session ends

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, gradients, transitions)
- Vanilla JavaScript (no frameworks/libraries)

## 🚀 Getting Started
1. Clone this repository
   ```bash
   git clone https://github.com/AliyaThabasum01/study-pomodoro.git
   ```
2. Open `index.html` in your browser — that's it, no build steps required.
